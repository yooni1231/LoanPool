☁️ Google Sheets 연동 설정

📋 설치 순서
  1. Google Sheets에서 새 스프레드시트 생성
  2. 상단 메뉴 확장 프로그램 → Apps Script 클릭
  3. 기존 코드 전체 삭제 후 제공된 Code.gs 내용 붙여넣기
  4. 💾 저장 → 배포 → 새 배포 클릭
  5. 유형: 웹 앱 / 실행: 나 / 액세스: 모든 사용자 → 배포
  6. 표시된 웹 앱 URL 복사 → 아래에 붙여넣기
✅ 연동 후 모든 변경사항이 자동으로 구글 시트에 저장됩니다.
